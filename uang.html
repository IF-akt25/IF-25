Konfigurasi Spreadsheet Diperlukan

Edit bagian CONFIG di dalam tag <script> pada file ini. Isi SPREADSHEET_ID dengan ID dari URL Google Spreadsheet Anda, lalu pastikan spreadsheet sudah dipublikasikan (File → Share → Publish to web).

// Temukan bagian ini di <script>
const CONFIG = {
  SPREADSHEET_ID : 'PASTE_ID_ANDA_DI_SINI',
  // ID ada di URL: /spreadsheets/d/INI_ID_NYA/edit
  SHEET_NAME : 'Sheet1', // nama tab
  DATA_MODE : 'gviz', // 'gviz' atau 'api'
};

Mode gviz tidak memerlukan API key — cukup publish spreadsheet ke web.
Mode api memerlukan Google Sheets API key dari Google Cloud Console.
